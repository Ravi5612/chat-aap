2.  **Run on Your Device/Simulator:**
    Once the server is running, you can choose your platform:
    *   **Android:** Press `a` in the terminal or run `npm run android`
    *   **iOS:** Press `i` in the terminal or run `npm run ios` (Requires macOS)
    *   **Web:** Press `w` in the terminal or run `npm run web`

3.  **Use the Expo Go App:**
    Scan the QR code displayed in the terminal using the **Expo Go** app on your physical device (available on Android and iOS).
